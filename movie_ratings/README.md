# Project: Movies Ratings Analysis (TMDb)

## About TMDb
The movie rating system has become an integral part of the film industry, providing audiences with a benchmark to gauge movie quality. Among the various platforms, **TMDb (The Movie Database)** is one of the most comprehensive and widely used. Established in 2008, TMDb aggregates detailed information about movies, including metadata, ratings, reviews, and more.

TMDb provides a **TMDb score**, which reflects audience ratings and popularity metrics. This scoring system allows users to quickly assess the general reception of a movie, explore trending films, and compare movies across genres and release years.

## About the Dataset
This dataset contains **1,000,000 movies** collected from TMDb. Each entry includes detailed information about:

- TMDb ratings and popularity scores  
- Movie title, release year, and genre  
- Directors, writers, and cast  
- Plot summaries and production details  
- Theatre and streaming release dates  
- Revenue, budget, and runtime  

The dataset enables exploration of relationships between **audience reception, popularity, and movie characteristics**, as well as trends across genres, years, and countries. It is ideal for performing detailed data analysis, visualisation, and machine learning tasks related to movies.
> **Note:** The full dataset is too large to host on GitHub. A **sample of the dataset** is included (`movies_sample.csv`) to demonstrate structure and analysis. The full dataset can be downloaded from [Kaggle]([https://www.kaggle.com/datasets/subhajournal/movie-rating](https://www.kaggle.com/datasets/asaniczka/tmdb-movies-dataset-2023-930k-movies))

## Project Focus
This project focuses on:  

- Writing SQL queries to explore, summarise, and analyse movie ratings and performance  
- Performing aggregations, joins, subqueries, and conditional logic  
- Using Excel or Power BI to create charts, pivot tables, and dashboards  
- Identifying trends such as top-rated movies, genre performance, and rating distributions  

## Dataset Columns

| Column Name      | Description |
|-----------------|-------------|
| id               | Unique identifier for each movie. (type: int) |
| title            | Title of the movie. (type: str) |
| vote_average     | Average vote or rating given by viewers. (type: float) |
| vote_count       | Total count of votes received for the movie. (type: int) |
| status           | The status of the movie (e.g., Released, Rumored, Post Production, etc.). (type: str) |
| release_date     | Date when the movie was released. (type: str) |
| revenue          | Total revenue generated by the movie. (type: int) |
| runtime          | Duration of the movie in minutes. (type: int) |
| adult            | Indicates if the movie is suitable only for adult audiences. (type: bool) |
| backdrop_path    | URL of the backdrop image for the movie. (type: str) |
| genres           | Genre(s) of the movie. (type: list of str) |
| directors        | Director names. (type: list of str) |
| writers          | Writers of the movie. (type: list of str) |
| cast             | Movie casting. (type: list of str) |
| popularity       | Popularity score calculated by TMDb. (type: float) |
| original_language| Original language of the movie. (type: str) |
| budget           | Budget of the movie in USD. (type: int) |
| production_companies | Production company names. (type: list of str) |
| production_countries | Production country names. (type: list of str) |
| spoken_languages | Languages spoken in the movie. (type: list of str) |
