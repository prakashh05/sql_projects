# Project: Food Delivery Data Analysis

## Description
This project uses the Food Delivery Dataset, part of the SQL Practice Dataset Series, which simulates a food delivery platform similar to services like Uber Eats or Deliveroo. The dataset contains realistic multi-table relational data designed to help learners practice intermediate SQL analysis and analytics concepts.

The dataset includes information on:
- Customers placing food delivery orders  
- Restaurants offering different cuisines  
- Menu items with prices  
- Order transactions and delivery times  
- Individual items purchased in each order  

This structure allows users to practice more advanced SQL concepts such as multi-table joins, revenue calculations, delivery performance analysis, customer behaviour analysis, and restaurant popularity analysis.

## Dataset Tables
The dataset contains 5 relational tables representing customers, restaurants, menu items, orders, and order items.

## Skills Practised
- **SQL:** Multi-table joins, aggregations, filtering, subqueries, revenue calculations, analytical queries  
- **Excel:** Pivot tables, charts, and dashboards to visualise order volumes, revenue, delivery performance, and popular menu items  
- **Data Analysis:** Examining customer behaviour, restaurant performance, ordering trends, and delivery metrics
