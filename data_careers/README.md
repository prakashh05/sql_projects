# Data Careers Analysis

## Description
This project uses a **Data Professionals Survey dataset**, based on responses from 630 participants, to explore trends in the data careers industry. The survey captures information on:

- Job roles within the data field
- Salary levels
- Work-life balance satisfaction
- Age and gender demographics

The dataset allows analysis of how compensation, satisfaction, and work preferences vary across different roles and demographic groups, providing insights into the state of data careers today.  

## Dataset Tables
The dataset contains key survey information, including:

- Respondent demographics (age, gender)
- Job roles and work experience
- Salary and work-style satisfaction
- Other factors influencing career choices in data roles

## Skills Practised
- **Power BI / Excel**: Creating interactive dashboards, charts, and pivot tables to visualise trends, comparisons, and key insights  
- **Power Query**: Data cleaning and transformation  
- **DAX**: Advanced calculations for analysis and metrics  
- **Data Analysis**: Examining patterns in salaries, job roles, work satisfaction, and demographic influences  
- **Data Visualisation**: Presenting findings clearly and interactively for effective storytelling  

This project strengthened my ability to analyse survey data, create dynamic dashboards, and derive actionable insights about data professionals’ career trends.
