# Project: Global Population Data Analysis

## Description
This project uses the **Global Population Dataset (CSV)**, which provides historical and projected population data from **1980 to 2050**. The dataset covers countries, states, provinces, and cities worldwide, allowing for analysis of population trends, growth rates, density, and regional comparisons.

The project focuses on:

- Writing SQL queries to explore, summarise, and analyse population data  
- Performing aggregations, subqueries, and ranking queries  
- Using Excel or Power BI to create visualisations, charts, and dashboards  
- Identifying trends such as countries with the fastest growth, population density patterns, and demographic shifts  

## Dataset Columns

| Column Name       | Description |
|------------------|-------------|
| place             | Geographical name of the entity (country, state, province, or city) |
| pop1980           | Estimated population in 1980 |
| pop2000           | Estimated population in 2000 |
| pop2010           | Estimated population in 2010 |
| pop2023           | Estimated population in 2023 |
| pop2024           | Projected population in 2024 |
| pop2030           | Projected population in 2030 |
| pop2050           | Projected population in 2050 |
| country           | Country to which the place belongs |
| area              | Total area including land and water (sq km) |
| landAreaKm        | Land area excluding water (sq km) |
| cca2              | Two-letter ISO country code |
| cca3              | Three-letter ISO country code |
| unMember          | Boolean flag for UN membership |
| netChange         | Net population change between 1980 and 2023 |
| growthRate        | Annual growth rate of population between 1980 and 2023 |
| worldPercentage   | Percentage of the world's population in the place |
| density           | Population density (people per sq km) |
| densityMi         | Population density (people per sq mile) |
| rank              | Rank of the place by population size (1 = most populous) |

## Skills Practised

- **SQL:** SELECT, JOIN, GROUP BY, aggregates, subqueries, ranking functions  
- **Excel / Power BI:** Pivot tables, charts, dashboards, conditional formatting  
- **Data Analysis:** Trend identification, growth rate calculations, population density analysis  
- **Documentation:** Writing clear explanations of queries and insights
