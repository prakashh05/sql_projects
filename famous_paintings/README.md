# Famous Paintings Analysis

## Description
This project uses the Famous Paintings Dataset, part of the SQL Practice Dataset Series, which simulates a global art museum ecosystem similar to institutions such as the Louvre and the Metropolitan Museum of Art. The dataset contains realistic multi-table relational data covering artists, artworks, museums, exhibition details, image references, and pricing information for artworks.

## Project Focus
- Writing SQL queries to explore, summarise, and analyse the data
- Performing multi-table joins, aggregations, filtering, and subqueries 
- Calculating artist performance, museum collection insights, artwork pricing trends, and subject distribution
- Using Excel to create charts, pivot tables to visualise trends and key metrics

## Dataset Tables
The dataset contains 8 relational tables representing:

- **artist** – Artist details including name, nationality, and style
- **canvas_size** – Canvas and artwork size/dimension information
- **image_link** – Image URLs and visual references for artworks
- **museum** – Museum information including name, location, and website URL
- **museum_hours** – Opening hours and operational timing for museums
- **product_size** – Pricing information including regular and sale prices of artworks
- **subject** – Artwork classification and thematic categories
- **work** – Artwork details including title, artist_id, and related metadata

## Skills Practised
- **SQL:** Multi-table joins, aggregations, filtering, subqueries, CASE statements, and analytical pricing queries
- **Excel:** Pivot tables, charts to visualise artist output, museum distribution, subject trends, and pricing behaviour

>Note: Dataset can be found at - https://www.kaggle.com/datasets/mexwell/famous-paintings
